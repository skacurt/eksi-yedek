eksi-yedek ("ekşi sözlük yedek okuyucu şeysi") is a PWA-compliant reader app for Eksi Sozluk backups. Eksi Sozluk backup files are XML files usually in a ZIP container with user entries and drafts.

# UI Style
- Use lowercase for all visible text unless specified by developer.

# Architectural rules
- Don't copy code if it's already available in another module. Reuse it instead.
- Don't add superfluous null checks for variables and properties that can't be null

# Coding style
- Don't use single-statements without braces after if, while, etc.
